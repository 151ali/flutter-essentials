# Introduction
## Packages (pub.dev)

> There are two types of packages we import usualy

1. Business logic packages
```dart
import 'package:http/http';
  getLocations() {
  return get('https://locations.xyz);
}
```

2. UI packages
```dart
import 'package:circle/circle.dart';
Container(
  child: Circle(
    color: Colors.red,
  ),
);
```
